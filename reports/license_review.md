# License Review

All current module rows use `verify_epaath_terms` and should be reviewed before redistribution.

Current working assumptions:

- OLE Nepal publicly exposes E-Paath activity directories and assets.
- Public availability does not by itself guarantee redistribution rights for derived datasets.
- Any final gold dataset release should document the exact usage basis for E-Paath-derived text.

Release rule:

No final dataset should be published until E-Paath usage and redistribution conditions are documented here.
