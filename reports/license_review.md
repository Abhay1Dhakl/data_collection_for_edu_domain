# License Review

All seeded sources currently use `verify_source_terms` or `verify_module_url_source_terms` in the registry and should be reviewed before redistribution.

Current working assumptions:

- CDC and CEHRD sources are official educational materials, but redistribution terms still need explicit review.
- OLE E-Paath modules require page-level verification before bulk extraction.
- OLE E-Pustakalaya exposes educational materials under platform terms that should still be checked at item level before release.

Release rule:

No source should move into the final dataset card until its usage conditions are documented in this report.
