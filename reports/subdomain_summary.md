# Subdomain Summary

- `mathematics`: highest current priority because public bilingual module directories are already visible.
- `science`: strong candidate for the same module-centric extraction workflow once bilingual assets are confirmed.
- `english_language`: useful for grades 1-8 where English content is available in module form.
- `nepali_language`: useful where bilingual module assets exist or where a Nepali counterpart can be paired at module level.
- `general_learning`: use for cross-subject early grade content if module structure is consistent.
- `early_grade_learning`: likely to yield short, instruction-heavy, manually reviewable EN-NE pairs.
